# 😊 AI Emoji Predictor

An AI-inspired web application built with **Python** and **Streamlit** that predicts the most suitable emojis based on user-entered text. The project uses Natural Language Processing (NLP) techniques through keyword matching to detect emotions, activities, and contexts, then suggests relevant emojis in real time.

---

## 🚀 Demo

> Enter a sentence like:

```text
I got an AI internship today!
```

**Output**

* 😊 Mood: Happy
* 📊 Confidence: 90%
* 🔍 Detected Keywords: AI, Internship
* 🎉 Predicted Emojis: 🤖 💼 🚀 🎉

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PRASAD1630/AI-Emoji-Predictor.git
```

### Navigate to the project folder

```bash
cd AI-Emoji-Predictor
```

### Install dependencies

```bash
py -m pip install -r requirements.txt
```

### Run the application

```bash
py -m streamlit run app.py
```

Open your browser and visit:

```text
http://localhost:8501
```

## 📖 How It Works

1. User enters a sentence.
2. The application preprocesses the text.
3. Keywords are matched against the emoji dataset.
4. Relevant emojis are predicted.
5. The application displays:

   * Mood
   * Confidence Score
   * Detected Keywords
   * Predicted Emojis
